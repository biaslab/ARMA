# ARMA

Custom ForneyLab.jl factor node for an _a_uto_r_egressive model with _m_oving _a_verage.

### Usage

The node can be added to ForneyLab by running

```julia
] dev git@github.com:wmkouw/ARMA.git
```

in the REPL and then import

```julia 
using ForneyLab
using ARMA
```
